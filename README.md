A bash script to create word lists of phone numbers.

### Requirements

- Crunch

### Usage

1. Edit the prefixes array inside the script.
2. Select a filename and set it the line below.
3. Run "chmod +x phonegen.sh".
4. Run "./phonegen.sh"
